Translation

With ML Kit's on-device translation API, you can dynamically translate text between more than 50 languages.

mlkit_translation_model_is_available

Try to install the language packages required for the translation model to work.

This function operates asynchronously, which means that it does not immediately return the requested result. Instead, upon completion of the task, it will trigger the Social Async Event.


Syntax:

mlkit_translation_model_is_available()


Returns:

N/A


Triggers:

Social Async Event

Key Type Description
type String The value "mlkit_translation_model_is_available"
success Real Whether there is a translation model available for the source
translator Real The translator unique identifier

mlkit_translation_translate

Translates the given input from the source language into the target language.

This function operates asynchronously, which means that it does not immediately return the requested result. Instead, upon completion of the task, it will trigger the Social Async Event.


Syntax:

mlkit_translation_translate(text)
Argument Type Description
text String The text to be translated


Returns:

N/A


Triggers:

Social Async Event

When a translation succeeds:

Key Type Description
type String The value "mlkit_translation_translate"
success Real The value 1
translator Real The translator unique identifier
text String The translated text


When a translation fails:

Key Type Description
type String The value "mlkit_translation_translate"
success Real The value 0
translator Real The translator unique identifier
error String The error message

mlkit_translation_close

Closes the translation unit and releases its resources.

Warning

Should be called when the translation unit is no longer needed.


Syntax:

mlkit_translation_close(translation_unit)
Argument Type Description
translation_unit Real The unique id of the translation unit to be closed.


Returns:

N/A
